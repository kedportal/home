Korpussuche

Suche über die KED-Suche

Die KED-Suche ist benutzerfreundlich und stärker auf die Bedürfnisse von Lehrkräften ausgerichtet. Eine Schritt-für-Schritt-Anleitung zur Registrierung für die KED-Suche finden Sie hier.

KED-Suche online

Die KED-Suche ist als Web-Anwendung unter folgender Adresse aufrufbar:

https://ked-suche.korpus-einfaches-deutsch.de/

KED-Suche auf dem eigenen Computer

Die KED-Suche ist auch als Desktop-Anwendung verwendbar. Aufgrund begrenzter Ressourcen können maximal 15 Nutzer die KED-Suche gleichzeitig online aufrufen. Wenn alle Plätze belegt sind, müssen Sie leider warten, bis ein Platz frei wird. Wenn Sie nicht warten möchten oder mehr Plätze benötigen, können Sie die KED-Suche auch auf Ihrem Computer installieren. Die folgenden Schritte beschreiben die Installation unter Windows und Linux/Ubuntu.

Docker installieren

Um die KED-Suche auf Ihrem Computer zu nutzen, benötigen Sie die das Programm Docker. Laden Sie Docker von der Docker-Website herunter und installieren Sie es. Folgen Sie dabei den Anweisungen für Ihr Betriebssystem.

KED-Suche herunterladen: Windows/MacOS

Öffnen Sie Docker Desktop und geben Sie im Suchfeld KED-Suche ein. Wählen Sie das Docker-Datei keddocker/ked-suche aus und klicken Sie anschließend auf Pull. Die Datei wird heruntergeladen. Sobald der Download abgeschlossen ist, klicken Sie auf Run. Beim ersten Ausführen müssen Sie unter Optional settings evtl. noch den sogenannten Port (3838) der Datei angeben (Screenshot) und erneut auf Run klicken.

Klicken Sie anschließend in Docker Desktop auf den aktiven Port oder rufen Sie folgende Adresse in Ihrem Internetbrowser auf: http://127.0.0.1:3838/. Die KED-Suche wird daraufhin im Internetbrowser geöffnet.

Um die KED-Suche wieder zu stoppen, klicken Sie in Docker Desktop neben der aktiven Anwendung auf Stop.

KED-Suche herunterladen: Ubuntu/Linux

Öffnen Sie ein Terminal und führen Sie die folgenden Befehle nacheinander aus, um die KED-Suche herunterzuladen und zu starten:

docker pull keddocker/ked-suche
docker run -p 3838:3838 keddocker/ked-suche

Navigieren Sie anschließend in Ihrem Internetbrowser zu folgender Adresse:

http://127.0.0.1:3838/

Sie stoppen die KED-Suche im Terminal mit der Tastenkombination Strg + C.

Suche über KorAP

Das KED ist über die Plattform KorAP des Leibniz-Instituts für deutsche Sprache in Mannheim online durchsuchbar. Eine Schritt-für-Schritt-Anleitung zur Registrierung auf KorAP finden Sie hier („Erster Schritt: Registrierung beim IDS“). Die Nutzung der KorAP-Plattform erfordert etwas Übung und ist stärker auf die Bedürfnisse von Forschenden ausgerichtet.

https://korap.ids-mannheim.de/instance/ked

Suche im DeReKo (via COSMAS II)

Das KED ist seit Anfang 2025 als Teil des Deutschen Referenzkorpus (DeReKo) am Leibniz-Institut für deutsche Sprache in Mannheim über die Anwendung COSMAS IIweb (https://cosmas2.ids-mannheim.de/cosmas2-web/) durchsuchbar.

Nach erfolgter Anmeldung / Registrierung finden Sie das KED dort im „W2 – Archiv der geschriebenen Sprache“ unter den „vordefinierten Korpora“ unter der Sigle „ked - Korpus Einfaches Deutsch, 2023 – 2024“.
